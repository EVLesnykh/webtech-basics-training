# WebTech Basics Training

Репозиторий с практическими заданиями и уроками по **основам веб-технологий**.  
Содержит решения домашних заданий и учебные материалы, которые помогут закрепить навыки фронтенд-разработки.

---

## Используемые технологии

- **HTML** — структура веб-страниц  
- **CSS** — оформление и стилизация  
- **JavaScript** — интерактивность страниц  
- **PHP** — небольшие серверные примеры (если применялись)
